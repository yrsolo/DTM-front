# Promo Comparison Checklist

Использовать после каждой итерации `/promo`.

- [ ] На экране есть один доминирующий фокус, а не несколько равносильных объектов.
- [ ] Секция не превратилась в набор одинаковых карточек.
- [ ] Screenshot ощущается как полноценная сцена, а не как картинка внутри панели.
- [ ] Акцентный цвет используется дозированно и не шумит на каждом блоке.
- [ ] Секция читается как один крупный narrative beat.
- [ ] Между секциями достаточно вертикального воздуха.
- [ ] Video, screenshot и CTA не спорят друг с другом за внимание.
- [ ] Страница ощущается как editorial landing, а не как fragments of dashboard UI.
- [ ] Текст и визуал работают как одна композиция, а не как два параллельных слоя.
- [ ] Следующая итерация уже понятна: видно, какой slice надо приближать дальше.
