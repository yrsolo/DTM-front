# CAM-WORKBENCH-AUTHORING-MODEL

## Objective

Define the authoring-facing model on top of parsed source nodes without coupling it to runtime inspection.

## In scope

- `AuthoringNode`
- `AuthoringValue`
- scopes: token, component, instance
- distinction between component definition editing and placement-level editing
- grouping, favorites, inheritance, value-source semantics

## Out of scope

- source graph parsing
- source patch generation
