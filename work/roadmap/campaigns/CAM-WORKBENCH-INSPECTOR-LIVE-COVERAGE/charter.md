# CAM-WORKBENCH-INSPECTOR-LIVE-COVERAGE

## Objective

Broaden verified semantic-target coverage across real mounted app surfaces without coupling package code to page-specific runtime.

## In scope

- expand app target registry for live surfaces
- verify `Alt+Click` and hierarchy navigation on mounted screens
- keep mode-gated or unmounted targets represented semantically even when not live

## Out of scope

- package-level navigation features
- focus-set UX
- ownership modeling changes
