# CAM-WORKBENCH-INSPECTOR-NAVIGATION

## Objective

Turn the inspector sidebar into a first-class semantic hierarchy browser for design navigation.

## In scope

- tree rendering from semantic targets
- expand/collapse state independent from selection
- semantic search that reveals ancestor paths
- cleanup of temporary flat target-browser UX

## Out of scope

- runtime value editing
- page-specific logic inside the package
- focus-set persistence details beyond what navigation needs
