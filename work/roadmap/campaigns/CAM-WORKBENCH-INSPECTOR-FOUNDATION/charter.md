# CAM-WORKBENCH-INSPECTOR-FOUNDATION

## Objective

Create the initial package boundary and app integration seam for a future-extractable local dev-only workbench inspector.

## In scope

- package skeleton
- public contracts
- app integration skeleton
- safe activation path
- docs and tracking

## Out of scope

- selection UX
- ownership binding UX
- editing runtime values
- backend or production wiring
