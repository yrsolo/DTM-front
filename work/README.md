# Work (process)

This folder contains **delivery/process** artifacts (not system documentation).

## Areas
- `work/now/` — the single active campaign/sprint (what is being executed now)
- `work/roadmap/` — planned campaigns with priorities (not started yet)
- `work/archive/` — historical campaigns (completed)

## Campaign IDs
Campaigns are identified as:
- `CAM-<NAME>-P##-T###` (campaign → phase → task)

Rules:
- Numbering is local inside a campaign.
- Keep tasks small and reversible.
- Evidence for decisions and results goes into campaign files.
