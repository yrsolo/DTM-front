# Promo Copy

Файл фиксирует рабочий copy для `/promo`.
Принципы:
- коротко, живо, по делу
- без рекламной истерики и без "объяснений для всех"
- если экран в драфте короткий, текст здесь тоже короткий
- иллюстрации и число экранов не меняем; тексты подстраиваем под существующую композицию
