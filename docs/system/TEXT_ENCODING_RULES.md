# Text Encoding Rules

Цель: больше не чинить “сломанный русский” вручную.

## Rules

1. Все текстовые файлы храним в UTF-8 + LF.
2. Русские строки в `tsx/ts`:
   - либо обычный UTF-8 текст,
   - либо `\uXXXX` escape, если редактор/терминал нестабилен по кодировке.
3. Запрещено сохранять файлы через shell-команды без явной кодировки.
   - Для PowerShell всегда указывать `-Encoding utf8`.
4. Перед деплоем запускать проверку:
   - `npm run check:mojibake`
5. Если проверка нашла типичные артефакты mojibake (латинские символы вместо кириллицы), исправление обязательно до коммита.

## Commands

```bash
npm run check:mojibake
```
